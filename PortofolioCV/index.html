  • Kehidupan Bersekolah

    Selama kurang lebih 10 tahun saya bersekolah di suatu daerah pelosok, pedalaman OKI. tak ada kendaraan bermotor disana. Hanya sebuah sepeda dan sampan sebagai alat transportasi sehari-hari. Ketika lulus SMP, saya pindah ke Jawa Tengah untuk melanjutkan SMA.

Saya mengambil jurusan IT karena saya menyukai komputer, dan ingin memperdalam kesenangan saya terhadap bidang tersebut. saat kuliah, saya juga berperan sebagai karyawan kontrak, dimana pada saat saya semester 6, kontrak pekerjaan saya habis, dan artinya saya juga harus mengubur impian menyelesaikan kuliah saya.

    Kandasnya pekerjaan dan kuliah saya, membuat saya juga kehilangan tempat tinggal saya (indekost). Beruntung, keluarga saya mau menampung saya untuk sementara. Saat itu, saya merasa sangat jatuh. Tapi tak lama kemudian, saya diberi kesempatan untuk melanjutkan ilmu saya dengan sekolah kursus. Saya merasa hidup kembali.

  • Membantu Skripsi

    Setelah saya selesai kursus, teman-teman seangkatan saya mulai mengerjakan skripsi. Kebanyakan teman saya adalah pekerja, yang tidak memiliki waktu banyak untuk menyelesaikan skripsi tersebut. Saya menawarkan diri untuk membantu mereka, sebagai pengisi waktu luang dan sebagai ilmu yang bisa saya kembangkan.

    Toko Online sebagai eksperient saya. Saya berhasil menyelesaikan satu program tersebut, meskipun tidak 100% original karya saya. Namun demikian, saya senang, karena saya mendapatkan banyak pelajaran. Sampai hari ini, saya pun benar-benar harus belajar lebih banyak lagi untuk bisa menjadi programmer yang baik.
